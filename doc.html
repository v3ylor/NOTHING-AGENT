  1. Users stake 100,000 $NAGNT to submit a formal protocol upgrade.
  2. The DAO votes over a 7-day epoch.
  3. If the proposal passes, the Agent receives the payload.
  4. The Agent immediately drops the payload and goes back to sleep.
